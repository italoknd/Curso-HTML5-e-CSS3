Texto Marcado

Podemos também adicionar textos com marcações utilizando a tag MARK

Mas como mudar a cor amarela do marcador?

Como origem, a cor amarela é escolhida como uma cor de marcação de texto, porém as vezes essa cor não combina muito com o site ou simplesmente você gostaria de trocar essa cor, e para tanto deve-se utilizar um pouco de CSS

Como fazer:

Basta digitar o parâmetro: style = "background-color: (Digite uma cor)", dentro da tag "Mark" e escolher a cor desejada para o seu marcador.

Ainda sobre o Mark:

Quando temos muitas marcações de texto num site fica um pouco inviável sair colocando varias cores diferentes em cada marcação de uma por uma, e para tanto temos que adicionar um comando que faça isso para todos os marks (lembrando que isto só serve para os marks que você quer que seja de outra cor que não seja o amarelo).

Teremos que ir até o Head (cabeçalho) do nosso documento e adicionar a tag style, dentro dessa tag nos vamos adicionar um seletor para mark desse jeito:

style

mark{
}

/style

E dentro desse seletor nos vamos colocar o parâmetro "background-color" e escolher uma cor para todos os seus marks.

Tag para textos "grandes" e "pequenos"

A tag usada para textos grandes é a tag "BIG", porém esta tag já é considerada obsoleta e logo, logo entrará em desuso, e a tag usada para textos pequenos é a tag "SMALL"

Tag para textos excluídos e inseridos:

A tag utilizada para

textos deletados

é a tag DEL. E para texto inserido é a tag INS.

Tag para textos sobescritos e subscritos:

Essas tags são muito importantes para a representação de algumas informações do tipo: 2 elevado ao cubo ou aquele 2 pequeno da fórmula da água, e para isso usamos:

A tag SUP na situação do 23, essa tag irá SOBESCREVER o 3 do expoente.

A tag SUB na situação do H2O, essa tag irá SUBSCREVER o 2 da fórmula.
