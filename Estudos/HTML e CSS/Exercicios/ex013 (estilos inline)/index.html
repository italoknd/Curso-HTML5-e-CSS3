O metódo INLINE:

O metódo inline do CSS serve para configurações pontuais nos sites, ou seja, em algumas linhas em especifico, é muito inviavel usa-lo para fazer várias operaçoes por todo o site, dá muito trabalho, principalmente se o site for grande, tiver muitas páginas e etc. Mas... algumas de suas funcionalidades estão descritas abaixo nas imagens, funcionalidades como:

Style - Color:


Este parametro é usado para mudar a cor de um texto em especifico, neste caso, usado em h1.

Style - Background-Color


Este parametro serve, bem como o nome diz, para mudar a cor do background do site, neste caso ele está sendo usado no body (corpo) do site.

Font Family


O Font Family, serve bem como o nome diz para mudar a cor da fonte de uma determinada área, neste caso, ele alterou a font de todo o texto presente dentro do body (corpo) do site, ja que ele está sendo usado dentro do body.
