1 - Código fonte / Pré-Formatação:

O comando "document.getElementById('teste') é um comando de JavaScript. Para uma sequência de caracteres que deixam as letras muito coladas usa-se a tag "< code >"

int n=1, n2=1, res; res = n1+n2; if(res==2){ printf("\n Ola mundo"); }

*A tag "< code >" não formata um txt como um código identado acima por exemplo, o txt entre as tags será exibido todo em uma linha horizontal. Para se corrigir isso, usa-se a tag "< pre >" cobrindo a tag e o txt contido dentro dele (ex a seguir). 

    
        
            
    int n=1, n2=1, res;

    res = n1+n2;
    
    if(res==2){
        printf("\n Ola mundo");
    }

CORRIGINDO A TABULAÇÃO: Acima foi apresentado a correção da identação do código, para fazer isso basta selecionar a parte do txt que deseja consertar (sem as tags) a tabulação e executar o CMD: SHIFT + TAB.
