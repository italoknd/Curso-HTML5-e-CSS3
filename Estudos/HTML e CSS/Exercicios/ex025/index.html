Anatomia de uma caixa

Border

A borda, é uma linha que cerca o seu conteúdo.

Padding

Padding no css serve para dar espaçamento em elementos como a própria borda (border), para que a mesma fique distante do conteúdo. Esta palavra também possui traduções como: acolchoamento e preenchimento

Margin

Usado para criar um espaçamento externo para que não haja conflito entre bordas

Outline

Outline é o contorno que se pode fazer entre o padding e o margin

Tipos de caixa

Existem dois tipos de caixa, o Box-Level e o Inline-Level.

  • Box-Level: Ocupa a largura (width) total da tela e tem um espaço só pra ele, ou seja, caso exista alguma caixa ocupando algum espaço na linha, ele automaticamente irá pular uma linha e criar a box-level.
  • Inline-Level: Diferentemente do modelo de caixa Box-Level, o Inline-Level não pula a linha automaticamente e nem ocupa a largura inteira da tela.
