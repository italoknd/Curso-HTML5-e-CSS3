Listas Ordenadas (OL)

O que são?

Listas ordenadas (OL) são listas em que a ordem faz diferença. São otimas para se otimizar tempo quando se trabalha com listas nas quais há muitos itens e numerações. E para utiliza-las basta usar a TAG < ol >. Cada item da lista é declarado pela TAG < li > que fica dentro da TAG "OL" . Segue o exemplo abaixo:

  1. Acordar
  2. Levantar da cama
  3. Escovar os dentes
  4. Tomar café

A numeração é organizada automaticamente, não precisando assim ser consertada caso haja erro na enumeraação da lista.

OBSERVAÇÃO: A partir do HTML 5 a utilização do /li tornou-se opcional, ou seja, isso significa que caso você queira apagar essa parte da tag do seu código agora você pode fazer isso sem se preocupar com erros.

Podemos ainda organizar o tipo da numeração que queremos mostrar na nossa lista e escolher um número que queremos que comece contando na lista.

Para escolher o tipo da numeração basta digitar a palavra "type" dentro da TAG < ol >. Ao fazer isso você terá uma série de opções para o tipo da numeração da sua lista. E para escolher um número para qual você quer que a lista comece a contar basta digitar "start" dentro da TAG < ol >, exemplo:

  1. Acordar
  2. Levantar da cama
  3. Escovar os dentes
  4. Tomar café



No exemplo acima o tipo da numeração mudou para algarismo romano e começou a contar a partir do número 5.

Listas não ordenadas(UN)

Neste tipo de lista não precisamos necessariamente enumerar uma sequencia de afazeres. E para criar uma lista nao ordenada (ou, unordered list) basta usar a TAG < ul >.

  • comprar pão
  • comer pão
  • tomar café
  • arrumar a casa

Podemos também personalizar as listas nao ordenadas inserindo o "type" dentro da TAG < ul >. Segue o exemplo:

  • comprar pão
  • comer pão
  • tomar café
  • arrumar a casa

No exemplo acima foi alterada a personalização da lista não ordenada.

Lista de definições:

A lista de definição pode ser acionada pela tag "< dl >". Esta tag é muito utilizada para descrever algo, como se fosse um dicionario. Existe ainda outras duas tags que acompanha a tag "< dl >", que são:
Tag < dt >: "Defintion term", esta tag é onde fica guardado o termo definição.
Tag < dd >: "Defintion description", esta tag é onde fica guardada a definição do termo em questão.
